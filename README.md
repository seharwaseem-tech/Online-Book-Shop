# OnlineBookShop

This repository contains a responsive online book shop project created for practice purposes.  

The website is developed using **HTML, CSS, and JavaScript**.  
It is a single-page design but includes multiple sections like **Home, Featured, Arrivals, Reviews, and Blogs**.  
The design is modern and interactive, allowing users to scroll through books horizontally and view dynamic content that moves automatically over time.  

## Project Overview
This practice project includes:

- **Home Section** – Introduction and main banner  
- **Featured Books** – Highlights selected books  
- **Arrivals** – Newly added books with horizontal scrolling  
- **Reviews** – Customer feedback displayed dynamically  
- **Blogs** – Latest posts or news related to books  
- **Responsive Design** – Works on desktop, tablet, and mobile devices  
- **Dynamic Interaction** – Books and sections move automatically to enhance user experience  

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  

## Purpose of This Project
The purpose of this project is to demonstrate:

- Building a visually appealing single-page website  
- Implementing horizontal scrolling and dynamic content  
- Responsive web design for multiple screen sizes  
- Practicing interactive UI elements using JavaScript
